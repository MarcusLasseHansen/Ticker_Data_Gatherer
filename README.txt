A small script for downloading options data and combining the data, this let's one create a small database of options data (longer than the yahoo finance intraday database).
Usage in a CLI: cd Ticker_Data_gatherer; python3 main.py
Requires yfinance, pandas, and collections to be installed. Furthermore, it is required for the script to be started in the folder it is in.

Note: Tickers are added in main.py, current ones are placeholders - not a proper portfolio!
